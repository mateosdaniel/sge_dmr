## 8. Piedra, papel o tijeras

```python
# Crea un programa que implemente el clásico juego de piedra, papel, tijeras, lagarto y spock.

import random

j1 = 0
j2 = 0
# Opciones a elegir, si no escribe una de estas solicita una nueva opción.
choices = ["piedra", "papel", "tijeras", "lagarto", "spock"]

        print("Opción no válida. Intenta de nuevo.")
        continue

    p2 = random.choice(choices)
    print("La máquina ha sacado:", p2)

    # Comprueba para cada caso si gana o pierde el jugador 1. Si gana se le suma un punto y sino se le suma a la maquina.
    match (p1, p2):
        case (p1, p2) if p1 == p2:
            print("Empate")
        case ("tijeras", "papel") | ("tijeras", "lagarto"):
            print("Jugador 1 gana esta ronda")

            j1 += 1
        case ("piedra", "lagarto") | ("piedra", "tijeras"):

        case _:

# Cuando uno de los 2 llegue a 5, sale del bucle e imprime el resultado.
```

```python
# Crea un programa que genere los primeros n números de la secuencia de Fibonacci.

n = int(input("Indica cuantos números de la secuencia de Fibonacci quieres: "))

if(n == 1):
    print("1")
elif(n == 2):
    print("1, 1" )
else:
    fib1 = 1
    fib2 = 1
    final = "1, 1"
    for i in range(n - 2):
        t = fib1
        fib1 = fib2
        fib2 += t
        final += ", " + str(fib2) 
    print(final)
```

## 10. Números primos
```python
# Crea un programa que solicite un número al usuario y le diga si es primo o no. Un número primo solo es divisible por 1 y por sí mismo.

n = input("Introduce un número: ")
while(not n.isdigit()):
    n = input("Ese no es un número válido, introduce un número válido: ")
n = int(n)

if n <= 1:
    es_primo = False
else:
    es_primo = True
    # Bucle desde 2 hasta la raiz cuadrada del número más uno porque el segundo parametro no está incluido.
    # Solo compruebo hasta la raiz cuadrada porque uno de los divisores siempre está en el rango menor o igual al número.
    for i in range(2, int(n**0.5) + 1):
        # Si es divisible, no es primo
        if n % i == 0:
            es_primo = False
            break

if es_primo:
    print(f"El número {n} es primo.")
else:
    print(f"El número {n} no es primo.")
```
